50–55 أسئلة أمن المعلومات على ملخص حماية البيانات B4 — إقامة البيانات داخل المملكة · RLS وتشفير عمودي · حجب الأدمن عن المحتوى السريري · سجلات إلحاقية · تسليم سرّ Bearer وتخزينه مشفَّراً. المخرج: أسئلة الامتثال مغلقة أو مسجّلة بمالك ومهلة. أمن المعلومات + Medify
55–60 اتفاق الخطوات التالية والملكيات — تحويل الاتفاق إلى التزامات باسم ومهلة. المخرج: الجدول أدناه معبّأ. مالك القرار التقني
المخرج الأهم من الجلسة: استكمال B1. اسم نظام الـHIS/EMR وقدراته هما ما يحدّدان أول موصّل عميق نبنيه؛ فالتكامل معياري — نواة FHIR R4 + مهايئ HL7 v2 (MDM/ORU للرفع وADT للديموغرافيا)، وموصّل لكل «نظام» لا لكل «مستشفى». تُقرأ B2–B4 قبل الجلسة: الجلسة للحسم لا للقراءة.
