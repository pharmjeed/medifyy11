خطة النشر المرحلية

sandbox ← UAT بقسم واحد ← production · لا انتقال بين مرحلتين إلا باستيفاء معايير خروج مقيسة وتوقيع مسؤول التكامل لدى المنشأة.

حدود النطاق (على كل المراحل): لا رفع إلى NPHIES ولا تنبؤ بقبول المطالبات — المنشأة ترفع المطالبة من نظامها، ومقياسنا «درجة اكتمال التوثيق» لا «احتمالية القبول» · DRG لا يُبنى إطلاقاً · التكامل معياري: نواة FHIR R4 + مهايئ HL7 v2، وموصّل لكل «نظام» لا لكل «مستشفى».

المراحل الثلاث

الحزمة الصادرة في كل المراحل: FHIR R4 Bundle نوع transaction = Encounter (class AMB) + Composition (LOINC 11488-4) + Condition[] + MedicationRequest[] + Procedure[] · POST بـapplication/fhir+json وBearer token.
